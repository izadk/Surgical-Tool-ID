How recognition works
1. Admins build a labeled tool class library
2. Anyone can upload a tool image to scan
3. Image + full class list → Claude Vision API
4. Claude returns ranked confidence for all classes
5. Results below threshold flagged as uncertain
6. Admins add more training notes to improve accuracy
